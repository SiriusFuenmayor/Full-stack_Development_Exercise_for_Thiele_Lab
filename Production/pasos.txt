- Luego correr el siguiente comando para correr los contenedores:

> docker-compose up

docker build -t nextjs-frontend .

docker run -d -t <IMAGE_NAME> --name <CONTAINER_NAME>
docker run -p 3000:3000 nextjs-frontend
